Key Concepts

Seven terms worth knowing before diving into the topic pages, in short form.

Onion Routing

Wrapping a message in multiple layers of encryption, one per relay, so each relay can only peel off its own layer, like peeling an onion.

Directory Authorities

Nine trusted server clusters that measure relay health hourly and publish a signed "network consensus" listing which relays are active.

Telescoping Circuit

A three-hop path (guard, middle, exit) built one encrypted hop at a time so no single relay ever learns the full path.

ntor Handshake

The Curve25519-based key exchange Tor uses to negotiate a separate symmetric key with each relay in a circuit.

Onion Service

A server reachable only through Tor via a cryptographic address, giving both visitor and host mutual anonymity.

Traffic Correlation Attack

An attack that matches the timing/volume of data entering a guard node to data leaving an exit node, potentially deanonymizing a circuit.

Post-Quantum Cryptography

Cryptographic algorithms designed to resist attacks from quantum computers, being blended into Tor's handshakes to defend against future decryption.
